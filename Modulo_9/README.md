# Tarefa 
Utilizando os conhecimentos que você adiquiriu neste curso, desenvolva novos scrapers:

* Desenvolva dois scrapers de dados, um com processamento concorrente e outro com processamento paralelo.
* Faça o scrape de pelo menos dez páginas
* Este exercício é livre, construa os scrapers da forma que preferir. É recomendado que tentam aplicar o conhecimento 
adquirido nos outros módulo para aprimorar os scrapers
